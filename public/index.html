开发说明

当前版本 v0.2.0

社区基础功能已上线，包括：注册登录、发帖评论、签到、点赞收藏、消息通知、排行榜系统。

技术架构

前端：Cloudflare Pages（全球CDN加速）
后端：Cloudflare Workers + D1数据库 + R2存储
成本：$0/月（全免费额度）

功能限制说明

  • 图片上传：单张最大2MB，自动压缩为WebP
  • 视频：暂不支持上传，可插入B站/抖音外链
  • 存储总量：10GB（免费额度），够初期使用

赞助目标

当前赞助：¥0 / 目标 ¥50/月（升级Workers付费版）
达成后解锁：视频上传、更大存储、更高请求上限

开发计划

  • ✅ Phase 1：社区基础功能
  • ✅ Phase 2：后端API + 前端重构
  • 🔄 Phase 3：消息系统+个人主页完善
  • ⬜ Phase 4：排行榜对接后端
  • ⬜ Phase 5：部署优化+安全加固
